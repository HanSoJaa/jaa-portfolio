# JAA PORTFOLIO PROJECT

This is a React project developed using Vite — a modern high-performance build tool with Hot Module Replacement (HMR) support and fast development cycles. This project features an interactive portfolio with 3D elements and smooth animations.

## 🛠️ How to Run This Project

1. **Clone this repository:**

   ```bash
   git clone https://github.com/username/jaa-portfolio.git
   cd jaa-portfolio
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Environment Configuration:**

   Copy `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```
   *Make sure the Supabase environment variables are correctly filled.*

4. **Run project locally:**

   ```bash
   npm run dev
   ```

   Then open your browser at: [http://localhost:5173](http://localhost:5173)

5. **Build for production (optional):**

   ```bash
   npm run build
   ```
